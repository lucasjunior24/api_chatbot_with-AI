# OpenAI: 

## ⚙️ Configuração do Ambiente

### Criando e Ativando o Ambiente Virtual

**Windows:**
```bash
python -m venv .venv
.venv\Scripts\activate
```

**Mac/Linux:**
```bash
python3 -m venv .venv
source .venv/bin/activate
```

### Instalação das Bibliotecas

```bash
pip install --no-cache-dir --upgrade -r requirements.txt
```
